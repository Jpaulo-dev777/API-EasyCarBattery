# # Easy Car Battery 

### Dominio
 O propósito App é transmitir a ideia de conveniência
 e facilidade na compra de baterias de carro

### Descrição
 Permite que você encontre, compre e agende a instalação de baterias de carro com facilidade e conveniência.
 Nossa missão é garantir que você nunca fique na mão por causa de uma bateria descarregada.Com uma ampla variedade de marcas e modelos, oferecemos a solução perfeita para o seu veículo, seja ele um carro, caminhão ou moto.
